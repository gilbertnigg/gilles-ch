# gilles-ch

Install frontend
- [Laravel Mix](https://laravel-mix.com  )
- [Tailwind CSS](https://tailwindcss.com )
```
npm i
```

Install backend
- [Kirby CMS](https://getkirby.com  )
```
cd public
composer install
```
---
run `npm` scripts

watch
```
npm run watch
```
production
```
npm run prod
```
